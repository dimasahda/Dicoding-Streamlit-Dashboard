## Setup Environment - Anaconda

## Setup Environment - Shell/Terminal
```
cd olist-dashboard
cd dashboard
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```
python3 streamlit run app.py  
```
